## 🚀 Quick Start

### For Python Developers
```bash
cd "problem_solving (python)"
python3 solution.py < input.txt
```

### For C++ Developers  
```bash
cd "problem_solving (C++)"
g++ -o solution solution.cpp
./solution < input.txt
```
